# ZC-R
## 목적
* `ZC-R`은 `Starcraft:Remastered`의 EUD 개발자를 위한 프로젝트입니다.
* 이 프로젝트는 문서형 사이트를 통해 [EDAC](https://cafe.naver.com/edac)의 한계를 보완하는 것입니다.
* 이 사이트는 사용자가 [EDAC](https://cafe.naver.com/edac)을 

## 프로젝트
### 완료
* 현재 ZC-R은 다음을 지원합니다.
* (아직 없음)

### 진행
* 현재 ZC-R은 다음을 진행 중입니다.
1. 문서 기여 방법 작성
2. Offset 분류
3. Offset 사용 예제 작성
4. 특이 유닛 정리
5. Order ID 정리

### 계획
* 미래 ZC-R은 다음 기능 지원을 목표하고 있습니다.
1. EUD 함수 분류
2. EUD 입문용 문서 작성
3. EUD 문서 작성
4. 유용한 EUD 기술 정리
5. EUD 기술 문서화

### 희망사항
* 미래 EUD 기술에 다음이 추가될 수 있도록 계획을 작성 중입니다.
1. CSV 지원 라이브러리
2. TUI/GUI 개발 지원 도구
3. EUD 라이브러리 개발 활성 방안
4. 게임 구성 분류 및 분류별 특화 라이브러리 개발
5. 동적 유닛
