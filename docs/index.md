# ZC-R
## 목적
* ZC-R은 Starcraft:Remastered의 EUD 개발자를 위한 프로젝트입니다.
* 이 프로젝트는 문서형 사이트를 통해 [EDAC](https://cafe.naver.com/edac)의 한계를 보완하는 것입니다.
## 프로젝트
### 완료
* 현재 ZC-R은 다음을 지원합니다.

### 진행
* 현재 ZC-R은 다음을 진행 중입니다.
1. Offset 분류
2. 문서 기여 방법
3. 함수 분류
4. EUD 문서 정리
5. EUD 입문용 문서 작성
6. 튜토리얼 작성
7. EUD 기술 정리

### 계획
* 미래 ZC-R은 다음 기능 지원을 목표하고 있습니다.
1. CSV 지원 라이브러리
2. TUI/GUI 개발 지원 도구
3. 게임 구성 분류
4. 게임 구성별 라이브러리 개발
5. 동적 타이머
6. 동적 유닛
