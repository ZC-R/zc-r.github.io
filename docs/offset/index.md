# 오프셋
## 오프셋 종류
### `.dat` 오프셋
dat 오프셋은 등 `.dat` 확장자를 사용하는 오프셋입니다. \
이 오프셋은 주로 구조 오프셋의 참조 데이터를 저장합니다. \
EUD가 아닌 경우 값이 변경되지 않습니다.

### 구조 오프셋
구조 오프셋은 기존의 오프셋 중 구조체로 구성된 오프셋입니다. \
게임 내의 각 객체를 가리킵니다. \
구조체 형태이기 때문에 주로 기준 오프셋으로부터 상대 값을 오프셋으로 나타냅니다.

### 데이터 오프셋
데이터 오프셋은 `.dat` 오프셋과 구조 오프셋을 제외한 다른 오프셋을 가리킵니다. \
다음의 하위 분류로 나뉩니다.
* 게임 데이터 (Starcraft Data)
* 플레이 데이터
* 플레이어 데이터
* 리플레이 데이터
